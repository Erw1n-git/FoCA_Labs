XML Filter output

Lists the test results of an XML filter.

To access this command...

Choose Tools - Macros - XML Filter Settings.


The test results of an import or export XSLT stylesheet are displayed in the XML Filter output window. If you want, you can also validate the filter output.

Validate

Validates the contents of the XML Filter output window.

The window splits into two areas and the results of the validation are displayed in the lower area.

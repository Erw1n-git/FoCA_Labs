Stepped Line Properties

In a chart that displays lines (Line type or XY type), you can choose to connect the points with steps instead of straight lines. Some options control the properties of those steps.

To access this command...

In the Chart Type dialog of a Line chart or XY chart that displays lines, choose Stepped in the Lines type drop-down, then click the Properties button.


Different step types

  • Start with horizontal line and step up vertically at the end.

  • Start to step up vertically and end with horizontal line.

  • Start with horizontal line, step up vertically in the middle of the X values and end with horizontal line.

  • Start to step up vertically to the middle of the Y values, draw a horizontal line and finish by stepping vertically to the end.

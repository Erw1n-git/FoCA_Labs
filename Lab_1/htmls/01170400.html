Data Source Title

Specifies a location for the address book file and a name under which the data source will be listed in the data source explorer.

Address Data Source Wizards - Data source title


Location

Specifies the location of the database file.

Browse

Specifies the location using a file dialog.

Make this address book available to all modules in LibreOffice

Registers the newly created database file in LibreOffice. The database will then be listed in the Data sources pane (CommandCtrl+Shift+F4). If this check box is cleared, the database will be available only by opening the database file.

Address book name

Specifies the data source name.

Grid

Sets the display properties of a grid.

To access this command...

Choose View - Grid and Helplines.


Display Grid

Displays or hides grid lines that you can use to align objects such as graphics on a page.

Snap to Grid

Automatically aligns objects to vertical and horizontal grid lines. To override this feature, hold down the Option key Alt key when you drag an object.

Grid to Front

Displays the grid lines in front of the objects on the slide or page.

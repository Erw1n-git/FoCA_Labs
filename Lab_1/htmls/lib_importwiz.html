Description

The ImportWizard library is used by the Document Converter wizard.

Its entry point is:

  • ImportWizard.Main.Main

Selecting the Document Converter wizard loads the following libraries in memory:

  • ImportWizard

  • Tools

Basic routine name conflicts may exist when multiple Basic libraries are loaded in memory.

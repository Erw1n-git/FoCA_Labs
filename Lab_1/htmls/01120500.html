Group Element Wizard: Create Option Group

Specifies a label for the option group.

To access this command...

In form design, click the Group Box icon on the toolbar
and use the mouse to create a frame - Last page of wizards.


Which caption is to be given to your option group?

Specifies the label for the option box. You will see the label of the group box displayed in the form. The text you enter here will correspond to the Label property of the group box.

Object Constants

Name

Type

Usage

Empty

Variant

The Empty value indicates that the variable is not initialized.

Null

null

Indicates that the variable does not contain data.

Nothing

Object

Assign the Nothing object to a variable to remove a previous assignment.

Example:


            SubExampleEmpty
                Dim sVar As Variant
                sVar = Empty
                Print IsEmpty(sVar) ' Returns True
            End Sub
            Sub ExampleNull
                Dim vVar As Variant
                MsgBox IsNull(vVar)
            End Sub
            Sub ExampleNothing
                Dim oDoc As Object
                Set oDoc = ThisComponent
                Print oDoc.Title
                oDoc = Nothing
                Print oDoc ' Error
            End Sub

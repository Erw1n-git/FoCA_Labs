Change Data Binding

Edit the data binding in the XForms Data Navigator.

Model

Select the name of the XForms model.

Item list

Displays the data binding for the selected form control. To change the data binding, select another item in the list click OK. To access the Add and Properties commands for an item, right-click the item.

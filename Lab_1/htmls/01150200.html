Searching in Japanese

Defines the search options for Japanese.

These commands can only be accessed after you enable support for Asian languages in LibreOffice - PreferencesTools - Options - Language Settings - Languages.

To access this command...

Choose LibreOffice - PreferencesTools - Options - Language Settings - Searching in Japanese.


Treat as equal

Specifies the options to be treated as equal in a search.

Ignore

Specifies the characters to be ignored.

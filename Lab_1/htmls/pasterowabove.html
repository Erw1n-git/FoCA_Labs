Paste as Row Above

Insert clipboard table data in a table as new rows instead of overwriting the content of the original cells of the target table.

To access this command...

Choose Edit - Paste Special - Paste as Rows above.

Right-click on the target table cell to open the context menu and choose Paste Special - Rows Above.

Close Window

Closes the current window. Choose Window - Close Window, or press CommandCtrl+F4. In the print preview of LibreOffice Writer and Calc, you can close the current window by clicking the Close Preview button.

If additional views of the current document were opened by Window - New Window, this command will close only the current view.

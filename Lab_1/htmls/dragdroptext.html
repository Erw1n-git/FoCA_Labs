Moving and Copying Text in Documents

  1. Select the text that you want to move or copy.

  2. Do one of the following:

    To move the selected text, drag the text to a different location in the document and release. While you drag, the mouse pointer changes to include a gray box.

    To copy the selected text, hold down Command Ctrl while you drag. The mouse pointer changes to include a plus sign (+).

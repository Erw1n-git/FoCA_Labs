Graphics Mode

Lists view attributes for the selected graphic object. The embedded or linked graphic object in the current file will not be changed, only the view of the object.

Graphics mode

Default

The view of the graphic object is not changed.

Grayscale

The graphic object is shown in grayscale. A color graphic object can become monochrome in grayscale. You can also use the color sliders to apply a uniform color to the monochrome graphic object.

Black and White

The graphic object is shown in black and white. All brightness values below 50% will appear black, all over 50% will appear white.

Watermark

The graphic object is raised in brightness and reduced in contrast so that it can be used in the background as a watermark.

Select Table

Specifies a table from the Seamonkey / Netscape address book source that is used as the address book in LibreOffice.

To access this command...

Address Data Source Wizards - Table selection


All tables from the first user profile will be registered for this data source in LibreOffice. You must specify one as the table that will be used in the LibreOffice templates.

List box

Specifies the table that is to serve as the address book for the LibreOffice templates.

You can make changes to the templates and documents at a later time by choosing Edit - Exchange Database.

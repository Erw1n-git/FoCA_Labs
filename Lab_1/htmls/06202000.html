Edit Custom Dictionary

Add and delete entries that are used for the Hangul/Hanja Conversion.

Book

Select the user-defined dictionary that you want to edit.

Original

Select the entry in the current dictionary that you want to edit. If you want, you can also type a new entry in this box. To move from the Original box to the first text box in the Suggestions area, press Enter.

Suggestions (max. 8)

Type a suggested replacement for the entry that is selected in the Original text box. The replacement word can contain a maximum of eight characters.

New

Adds the current replacement definition to the dictionary.

Delete

Deletes the selected entry.

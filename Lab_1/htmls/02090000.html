Select All

Selects the entire content of the current file, frame, or text object.

To select all of the cells on a sheet, click the button at the intersection of the column and row header in the top left corner of the sheet.

To access this command...

Choose Edit - Select All.

CommandCtrl+A

Select All


To select all of the sheets in a spreadsheet file, right-click the name tab of a sheet, and then choose Select All Sheets.

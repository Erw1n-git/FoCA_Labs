Compatibility

Defines compatibility options for LibreOffice Calc.

To access this command...

Open a spreadsheet document, choose LibreOffice - PreferencesTools - Options - LibreOffice Calc - Compatibility.


Key bindings

The following table shows what actions are associated with what key bindings for the two key binding types (Default and OpenOffice.org legacy):

Key binding

Default

OptionAlt+Down Arrow

Where the actions are:

  • delete contents - launch the Delete Contents dialog.

  • delete - immediately delete the cell content, without the dialog.

  • fill down - fill cell content downward within selection.

  • data select - launch the Selection List dialog.

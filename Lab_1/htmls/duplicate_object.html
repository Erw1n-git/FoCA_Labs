Duplicating Objects

You can create duplicate or multiple copies of an object. The copies can be identical or can differ in size, color, orientation and location.

The following example creates a stack of coins by making multiple copies of a single ellipse.

  1. Use the Ellipse tool to draw a solid yellow ellipse.

  2. Select the ellipse and choose Edit - Duplicate.

  3. Enter 12 as Number of copies.

  4. Enter a negative value for the Width and Height so that the coins decrease in size as you go up the stack.

  5. To define a color transition for the coins, select different colors in the Start and End boxes. The Start color is applied to the object that you are duplicating.

  6. Click OK to create the duplicates.

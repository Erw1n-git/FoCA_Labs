HTML Export - Page 5

Defines a button style for navigation through the presentation slides.

This page is not visible if you have unmarked the Create title page check box, or if you have selected either automatic or WebCast export.

To access this command...

LibreOffice Draw or LibreOffice Impress menu File - Export, select HTML file type, page 5 of the wizard.


Select button style

Specifies whether you want to insert navigation buttons in your presentation. You can also select the style of the buttons.

Text only

Inserts only text hyperlinks instead of buttons.

Selection field

Displays the available button styles. Click on a button style to select it.

Resetting Font Attributes

  • You can quickly exit manual formatting by pressing CommandCtrl+Shift+X. For example, if you have pressed CommandCtrl+B to apply the bold typeface to the text that you type, press CommandCtrl+Shift+X to return to the default character format of the paragraph.

  • To reset all direct formatting of existing text, select that text, then choose the menu command Format - Clear Direct Formatting.

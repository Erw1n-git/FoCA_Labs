Preview in Web Browser

Creates a temporary copy of the current document in HTML format, opens the system default Web browser, and displays the HTML file in the Web browser.

To access this command...

Choose File - Preview in Web Browser.


The HTML formatted copy is written to the temporary files folder that you can select in LibreOffice - PreferencesTools - Options - LibreOffice - Paths. When you quit LibreOffice, the HTML file will be deleted.

You can set the HTML export filter options by choosing LibreOffice - PreferencesTools - Options - Load/Save - HTML Compatibility.
